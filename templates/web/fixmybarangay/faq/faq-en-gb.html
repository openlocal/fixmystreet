Frequently Asked Questions

What is FixMyBarangay?
FixMyBarangay is a site to help people report, view, or discuss local problems they’ve found to the city departments. The pilot project lets you report the problems by SMS (or dropping into the barangay center).
What areas are covered?
The pilot project is just running in two barangays in Cebu – Luz and Basak San Nicolas.
What sort of problems should I report with FixMyBarangay?
The FixMyBarangay pilot project is just for potholes (and similar road surface problems) and broken streetlights.
What isn’t FixMyBarangay for?
FixMyStreet is not a way of getting in touch with Cebu City Hall for other problems – please use FixMyBarangay only for potholes and streetlights while the pilot project is running..
Staff only: How do I use the site?
If you have a staff login, you can view problems already reported in that area, or report ones of your own simply by clicking on the map at the location of the problem.
How are the problems solved?
Problems sent by SMS are initially handled by the barangay staff – they'll confirm the problem and report it to the relevant department automatically. Once it's in the department's problem-tracking system, they'll get onto it as soon as they can.
Is it free?
The site is free to use, yes. The pilot project has been funded by the World Bank.

Reporting

To make a new report, start by clicking on the button for either barangay BSN or Luz.

  • To create a standalone report simply click on the map where the problem is.
  • To create a report from a text message log into Message Manager (see below), then select the message you want to use (click on the radio button). When the message has gone green, you can click on the map and the message will automatically be used to create the report. You can edit the text, but that report will be associated with that incoming message.

Once you've placed the pin, but before you press Submit, you can drag or reposition the pin if you want to change its position.

Choose the category for this report ‐ obviously, this helps FixMyBarangay decide to which department the report will be sent.

How do I see the SMS messages people have sent?
When you look at the barangay problems, click on Problems via text (or, if you're viewing a specific report, click Show messages) to see the currently available messages. You'll need a username and password for the Message Manager (your manager will have told you what these are). Note that you can only see messages that have not yet been used to create a report (and also their replies).
Is my Message Manager username and password the same as the ones I use for FixMyBarangay?
No, it's a separate account (some FMB users don't have any access to the SMS messages). Your manager will tell you what your login for Message Manager system is, if you have one.
I tried to work with a message but it said "Someone is working with that message right now!" What's up?
This means another staff member has a "lock" on the message. To prevent you both creating a report, or sending a reply, we use a first-come-first-serve locking mechanism. Collisions like this are rare, but if they do happen, try again a few minutes later because the lock may have expired by then.
Can I delete a message?
Yes. Click on the red cross when you hover over it (although really you're only hiding it so it never gets shown within FixMyBarangay). You need a manager's account in the Message Manager to be able to see any messages that have been hidden, so hiding them is as good as deleting them.
It looks like an incoming message is wrongly shown as a reply. Can I fix it?
FixMyBarangay tries to recognise SMS replies, and automatically matches them to the right thread. If it gets this wrong you can click on the detach button. The message will then be a new "available" message. If Message Manager got it wrong the other way round, and missed a reply, you can fix this too but you need to log into Message Manager with a manager's account to do it.
How do the replies get back to the original sender?
Although you're replying from within FixMyBarangay, the replies are sent by SMS. There may be a delay of a several minutes between the moment you press reply and the message's arrival.
How can I tell the original sender their problem has been fixed?
Go to the report. If the report is marked "this report was submitted by SMS", you can send a reply. Click on Show Messages. When the messages appear, click on Show messages for this report. Hover over the message you want to send the message to, and click Reply.
